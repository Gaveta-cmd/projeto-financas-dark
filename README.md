# 💰 Personal Finance Dashboard — Dark Edition

> **⚠️ PROJETO EM DESENVOLVIMENTO:** Este sistema está sendo construído ativamente para fins de aprendizado e organização financeira pessoal. Funcionalidades e interface estão em constante evolução.

Um sistema de gestão de finanças pessoais focado em estética **High-End Tech**, desenvolvido para rodar localmente. O projeto utiliza uma interface baseada em um Design System minimalista com cores profundas e animações fluídas.

## 🚀 Status do Projeto
Atualmente em: **Fase de Implementação de Componentes e Lógica Financeira**.

## 🛠️ Tecnologias Utilizadas
- **React.js** (Vite)
- **Tailwind CSS** (Estilização)
- **Framer Motion** (Animações de interface)
- **Lucide React** (Ícones)
- **Git/GitHub** (Versionamento e Backup)

## 🎨 Design System & Estética
O projeto segue uma identidade visual rigorosa inspirada no estilo "Linear/Vercel":
- **Background:** `#000000` (Pure Black).
- **Accents:** `#ef233c` (Electric Red) para destaques e estados críticos.
- **Surfaces:** Zinc-900 e Zinc-800 com bordas tracejadas (dashed).
- **Animações:** Implementação de `fadeSlideIn` e `columnReveal` para uma experiência de usuário premium.
